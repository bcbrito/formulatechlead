## 📋 Sobre

**Fórmula Tech Lead** é um programa de mentoria personalizado e estruturado que acelera a evolução profissional de desenvolvedores. Oferecemos uma jornada clara e estratégica, com acompanhamento individual, feedback contínuo e métodos comprovados para transformar sua trajetória técnica em liderança de alto impacto.

Seja você um iniciante na carreira ou um senior buscando a transição para liderança técnica, temos um programa adequado para sua etapa.

---

## 🎯 Programas Oferecidos

### 🟢 **Dev Junior**
**Para:** Iniciantes em desenvolvimento  
**Objetivo:** Dominar fundamentos e alcançar autonomia técnica básica

- Lógica de programação
- Estrutura de dados
- Fundamentos de arquitetura
- Frontend e Backend
- Banco de dados

---

### 🔵 **Dev Pleno**
**Para:** Devs com 1-3 anos que buscam a primeira promoção  
**Objetivo:** Sair do nível Júnior e conquistar autonomia, consistência e reconhecimento

- Trilhas práticas estruturadas
- Desenvolvimento técnico e comportamental
- Mentoria estratégica
- Preparação para promoção
- Aumento de empregabilidade

---

### 🟣 **Dev Senior**
**Para:** Devs com 3-6 anos em busca de especialização  
**Objetivo:** Dominar arquitetura, tomada de decisão técnica e liderança de soluções

- Arquitetura de solução avançada
- System design e design docs
- Decisões técnicas estratégicas
- Ownership e alto impacto
- Reconhecimento como referência técnica

---

### 🟡 **Tech Lead**
**Para:** Devs Sênior com 8+ anos buscando liderança técnica  
**Objetivo:** Fazer a transição segura para Tech Lead com autoridade técnica e impacto

**Etapa 1 — Diagnóstico e Fortalecimento Técnico**
- Consolidar arquitetura de solução
- System design e design docs
- Bancos de dados avançado
- DevOps, SRE e Platform Engineering
- Qualidade e segurança

**Etapa 2 — Desenvolvimento de Liderança**
- Estilo de liderança pessoal
- Engajamento de times
- Comunicação assertiva
- Marketing pessoal
- Presença como referência

**Etapa 3 — Gestão de Produto e Projetos**
- Agilidade e frameworks
- Tech metrics (KPIs, OKRs)
- Colaboração com produto
- Visão sistêmica
- Influência organizacional

**Etapa 4 — Liderança de Times**
- Team Topologies
- Gestão de conflitos
- 1:1s e development pessoal
- Entrevistas e contratação
- Onboarding e rituais

## 🚀 Como Começar

### Pré-requisitos
- Node.js 18+ instalado
- npm ou yarn
- Git

### Instalação

```bash
# Clone o repositório
git clone https://github.com/brunocaetanobrito/formulatechlead.git
cd formulatechlead

# Instale as dependências
npm install --legacy-peer-deps

# Inicie o servidor de desenvolvimento
npm run dev
```

O site estará disponível em `http://localhost:3000`

### Build para Produção

```bash
# Build otimizado
npm run build

# Preview local do build
npm run preview
```
